# Together
**Together** – приложение для построения здоровых семейных отношений.

## Запуск бэкэнда (без развертывания модели)

1. Перейти в папку backend (cd backend)
2. Создать .env файл и установить переменные окружения для Docker Compose
3. docker compose up -d (запустить в папке backend)

## Запуск бэкэнда (c развертыванием модели)

1. Создать .env файл и установить переменные окружения для Docker Compose
2. docker compose up -d (запустить в корне проекта)
